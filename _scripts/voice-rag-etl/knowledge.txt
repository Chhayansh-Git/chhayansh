Project: AI Wedding Image Culling System
Chhayansh developed an automated computer vision pipeline for Fotographiya to curate massive raw photographic datasets. The system ingests event photos and filters them through multiple AI stages. Stage 1 uses Laplacian variance algorithms to discard blurry images. Stage 2 utilizes YOLOv8 to ensure human subjects are present, coupled with MTCNN for facial landmark analysis to discard photos with closed eyes or poor expressions. Stage 3 handles semantic duplication by converting images into vector embeddings using OpenAI's CLIP ViT-L/14; it groups identical shots regardless of camera movement by calculating cosine similarity. The entire inference architecture is orchestrated via Python and FastAPI to handle heavy concurrent tensor operations.
GitHub Link: https://github.com/chhayanshporwal

Project: 3D Calcium Imaging Denoising (AI4Life International Grand Challenge)
Chhayansh secured Global Rank 2 in the AI4Life International Grand Challenge. He developed a production-ready biological image restoration pipeline to denoise live-cell calcium imaging volumetric video. Because clean "ground truth" data does not exist for this cellular microscopy, he implemented a 3D Noise2Void (N2V) self-supervised neural network. Built using PyTorch, the model learns to reconstruct biological structures by predicting masked pixels from surrounding noisy data. He optimized high-dimensional data loaders and spatial tensor augmentations to process massive multi-frame TIFF sequences within GPU memory limits.
